# 🗺️ Mapty

A modern workout tracking application built with **HTML5**, **CSS3**, and **Vanilla JavaScript**. Mapty allows users to log running and cycling workouts on an interactive map using the browser's Geolocation API and Leaflet.js.

## 🌐 Live Demo

🔗 **Live Website:** https://mapway.netlify.app/

## ✨ Features

- 📍 Detects your current location using the Geolocation API
- 🗺️ Interactive maps powered by Leaflet.js
- 🏃 Log running workouts
- 🚴 Log cycling workouts
- 📝 Store workout details (distance, duration, cadence, elevation)
- 📌 Display workout markers on the map
- 📋 View workouts in a sidebar list
- 🎯 Click a workout to navigate to its location
- 💾 Persist workout data using Local Storage
- 📱 Responsive and user-friendly interface

## 📖 Usage

1. Allow location access when prompted.
2. Click anywhere on the map to add a workout.
3. Choose the workout type:
   - 🏃 Running
   - 🚴 Cycling
4. Enter the workout details.
5. Save the workout.
6. Click any workout in the sidebar to move the map to its location.
